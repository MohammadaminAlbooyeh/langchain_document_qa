# LangChain Guide

## What is LangChain?

LangChain is a framework for developing applications powered by language models.

## Key Concepts

### Chains
Chains combine LLMs with other components like prompts, memory, and tools.

### Retrieval-Augmented Generation (RAG)
RAG combines retrieval of relevant documents with LLM generation for accurate answers.

### Agents
Agents use LLMs to decide which actions to take and in what order.
